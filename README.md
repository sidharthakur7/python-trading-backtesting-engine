Python Trading Backtesting Engine

📌 Overview

This project implements a basic trading backtesting engine using Python.
It simulates buy and sell signals on historical stock data and evaluates strategy performance.

⚙️ Features

- Signal-based trading (Z-score strategy)
- Portfolio simulation
- Trade tracking
- Performance metrics:
  - Total Return (%)
  - Win Rate
  - Number of Trades
  - Average Profit per Trade

🛠️ Tech Stack

- Python
- Pandas
- NumPy
- yFinance

📊 Strategy

- Buy when Z-score < -0.75
- Sell when Z-score > 0.75

- Add equity curve visualization
- Add drawdown analysis
- Improve strategy logic
